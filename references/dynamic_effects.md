为一名计算机专业的学生，你的个人网页不仅是展示经历的简历，更是你的**技术审美**和**产品思维**的缩写。

针对你“AI Agent + 多元爱好”的背景，我为你设计了一套兼具**极简主义**与**硬核交互**的方案。这套方案的核心理念是：**让网页本身就像一个正在运行的 Agent，而你的爱好则是其抓取到的多维数据。**

---

## 1. 核心主题：AI Agent 智能交互态
**意图：** 体现你对 Agent 架构的理解，将网页变成一个“有意识”的系统。

* **“自主意识”光标 (Autonomous Cursor)：**
    * **效果：** 你的鼠标指针不再是普通箭头，而是一个带有延迟跟随效果的**粒子光点**。当光标移动到文字上方时，模拟文字扫描的效果，用一道白色的光线扫过整段文字。
    * **技术：** 使用 `Canvas API` 或 `Three.js` 实现粒子动力学。
* **指令交互式导航 (Command-Line Interface)：**
    * **效果：** 在页面角落保留一个极简的输入框。用户可以输入命令如 `/travel` 或 `/music`，网页会像 Agent 执行任务一样，平滑滚动并自动展开对应板块。
    * **高级感：** 模拟流式输出（Streaming）的效果，文案逐字蹦出，展示 Agent 的推理过程。

---

## 2. 旅游与摄影：光影空间感
**意图：** 将静态照片转化为动态的探索体验。

* **“镜头焦距”滚动交互：**
    * **效果：** 当用户滚动到摄影板块时，背景图片最初是模糊的（Bokeh 效果）。随着滚动，图片逐渐变得清晰，仿佛相机在**自动对焦**。
    * **技术：** CSS `backdrop-filter: blur()` 与滚动驱动动画（Scroll-driven Animations）结合。
* **足迹连线地图 (Path Animation)：**
    * **效果：** 放置一张极简的 3D 地图。当你悬停在某个城市名上时，地图上会自动绘制一条从你当前所在地（例如：长沙）出发的**弧线**，并弹出一张该地的摄影缩略图。
    * **交互：** 模拟 $O(log n)$ 搜索算法的视觉感，线条绘制带有电信号流动的质感。

---

## 3. 音乐与金融：频率与曲线
**意图：** 将这两种看似无关的领域，通过“波动”这一共性联系起来。

* **音频采样数据背景：**
    * **效果：** 背景中有一条极细的、近乎透明的**正弦波曲线**。
    * **交互：**
        * **音乐态：** 播放音乐或鼠标移动时，曲线波动频率加快，模拟音频频谱。
        * **金融态：** 点击金融板块，曲线平滑切换成 K 线图或复利增长曲线（Exponential Growth），并显示实时的模拟金融指标。
    * **数学美感：** 动态展示复利公式：$$A = P \cdot (1 + r)^t$$，数值随滚动实时跳变。

## 4. 技术方案推荐

为了实现上述“高级感”，建议使用以下技术栈：

| 维度 | 推荐工具 | 理由 |
| :--- | :--- | :--- |
| **基础框架** | **Next.js** | 极佳的 SEO 和服务端渲染，适合个人品牌。 |
| **动效库** | **Framer Motion** | 声明式动效，处理进入/退出动画非常优雅。 |
| **3D 渲染** | **Three.js / React Three Fiber** | 实现 3D 地图或粒子 Agent 必选。 |
| **着色器** | **GLSL** | 如果想做极致的“流光”或“扫描”效果，着色器是加分项。 |

---

## 设计建议
**切记“少即是多”：** 作为一个准程序员，不要把所有动效堆叠在一起。建议保持背景为纯净的深色（Dark Mode）或极简灰，通过**高饱和度的局部色彩**（如 Agent 常用的人造蓝、金融常用的增长绿）来引导用户的注意力。
